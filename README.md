# Open Label Psilocybin TMS-EEG study 

This folder contains OPTE experimental, preprocessing and analysis scripts. 
Tasks include cognitive (cognitive control), resting state EEG (eyes open-closed), task fMRI and TMS-EEG measurements (single pulse, LICI, SICI, ICF protocols) at baseline, under acute psilocybin, 1-week, 2-week, 1-month post-psilocybin administration.
